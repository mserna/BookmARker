# BookmARker
AR Bookmarker

|--- Overview of the App ---|

BookmARker is an iOS application that auto-captures, auto-recognizes and creates digital bookmarks for users with reading material. BookmARker also allows users to store bookmarks in a book catalog library. All bookmarks will also have the feature to include notes for the specific page. BookmARker is an easy and helpful tool for avid readers, students, teachers, and/or other job proffessions that require easy bookmarking.
